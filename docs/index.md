## Project Overview

**FeedMe** is a food delivery platform inspired by FoodPanda, allowing users to browse restaurants, build a cart, and order food for delivery. The project was developed over two agile iterations as part of CP3407.

| Item | Details |
|------|---------|
| **Subject** | CP3407 Advanced Software Engineering |
| **University** | James Cook University (JCU) |
| **Year** | 2026 |
| **Methodology** | Agile (XP-inspired, 2 iterations) |
| **Frontend** | Next.js 14 / TypeScript |
| **Backend** | Django 5 / Python |
| **Database** | SQLite (development) |
