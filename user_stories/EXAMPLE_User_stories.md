# Initial User Stories Backlog

## Backlog

- As a user, I want to create an account so that my personal information is saved.  
  Priority: Must-have | Estimate: 2 days

- As a user, I want to log in securely so that my account is protected.  
  Priority: Must-have | Estimate: 1 day

- As a user, I want to browse available food options so that I can choose meals easily.  
  Priority: Must-have | Estimate: 3 days

- As a user, I want to place an order so that I can receive food.  
  Priority: Must-have | Estimate: 3 days

- As a user, I want to track my order status so that I know when it will arrive.  
  Priority: Should-have | Estimate: 2 days

- As an admin, I want to manage food listings so that the system stays up to date.  
  Priority: Could-have | Estimate: 2 days
